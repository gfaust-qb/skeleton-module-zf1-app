# skeleton-module-zf1-app
Skeleton to setup zendframework 1 modules like described in [https://framework.zend.com/manual/1.12/en/project-structure.filesystem.html]().

## Structure
* `configs/`
  * `application.ini`
* `controllers/`
  * `helpers/`
* `forms/`
* `layouts/`
  * `filters/`
  * `helpers/`
  * `scripts/`
* `models/`
* `services/`
* `views/`
  * `filters/`
  * `helpers/`
  * `scripts/`
* `Bootstrap.php`
